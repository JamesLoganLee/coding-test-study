## 📝 배운 것 메모 (헷갈릴 때 여기 보기)

**입출력**
- `n = int(input())` — 입력은 항상 '문자'라 숫자로 쓰려면 `int()`
- `a, b = map(int, input().split())` — 한 줄에 여러 숫자

**조건문**
- 짝수 판별: `num % 2 == 0` (2로 나눈 나머지가 0)
- `if/for/else` 줄 끝엔 **항상 `:` (콜론)**
- `==`(같다·판단) / `=`(넣다·대입) / `!=`(같지 않다)
- 따옴표 `" "` 안은 '글자', 밖은 '계산'

**반복문**
- `for i in range(1, n+1):` — 1부터 n까지 하나씩
- `answer += i` — answer에 계속 더해 쌓기 (누적 패턴)

**테스트**
- `print(solution(4))` — 함수를 직접 불러 결과 확인
- 노란 글씨 = 경고(무시 가능) / 빨간 글씨 = 오류(고쳐야 함)
