## 📝 배운 것 메모 (헷갈릴 때 여기 보기)

### 1주차 — 기초 문법
- `n = int(input())` — 입력은 '문자'라 숫자로 쓰려면 `int()`
- 짝수 판별: `num % 2 == 0`
- `if/for/else` 줄 끝엔 **항상 `:` (콜론)**
- `==`(같다·판단) / `=`(넣다·대입) / `!=`(같지 않다)
- 따옴표 `" "` 안은 '글자', 밖은 '계산'
- `for i in range(1, n+1):` — 1부터 n까지 / `answer += i` — 누적

### 2주차 — 리스트
- **리스트** = 여러 값을 담는 상자 `[10, 20, 30]`. 맨 앞은 **0번**부터 (`numbers[0]`)
- `for x in numbers:` — 리스트에서 값을 직접 하나씩 꺼내기
- **기준값 갱신 패턴** (최대·최소 찾기의 핵심):
  ```
  기준 = 첫 값
  for x in 리스트:
      if x가 더 좋으면:
          기준 = x
  ```
- **편리한 도구 3형제**: `sum()`(합) · `len()`(개수) · `max()`/`min()`(최대·최소)
- **엣지 케이스**: 빈 리스트 대비 `if not numbers: return None` (안 하면 `numbers[0]`에서 오류)
- **함수의 재료(매개변수)는 밖에서 받는 것** — 함수 안에서 `numbers = [...]`로 다시 정하면 안 됨!
